== Description ==

Re-positions your wordpress admin menu from the left side to the top. With smooth dropdown display. Have a less cluttered admin area for you or your clients to use.

The plugin does what it says . Floats your menu to the top bar on your wordpress installation. Is now responsive


== Installation ==

1. Install the WP Float Admin Menu either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the plugin.
3. Your menu should now have the new layout.
4. Access the menu admin via Appearance :: WP Float menu
